# S.O.L.I.D. FizzBuzz
## Requirements:
- PHP ^7.3

#### After cloning the repository:
`cd solid-fizzbuzz`
`composer install`

### Usando o PHP e Composer local
<b> Testes </b>

`composer run test`

### Running in Docker:
<b> Testes </b>

`docker-compose up`

<b> Detached </b>

`docker-compose up -d`
`docker exec -it solidfizzbuzz bash`
`composer run test`
